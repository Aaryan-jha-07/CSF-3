# Personal Portfolio Website

A responsive personal portfolio website designed to showcase my skills, projects, and contact information. This project was developed as part of the "Computer Science Fundamentals & Career Pathways" course assignment to demonstrate proficiency in HTML, CSS, and version control using Git and GitHub.
